# AI-Based Adaptive Personal Tutor System

This project is an AI-powered personal tutor that helps students manage their studies by providing
personalized learning plans, quizzes, and feedback based on their performance.

## Features
- Conversational learning interface
- Quiz-based performance evaluation
- Student level classification using Random Forest
- Personalized study plan generation
- Continuous adaptive learning

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Machine Learning: Random Forest (Scikit-learn)
- NLP: NLTK, TF-IDF
- Database: SQLite

## Use Case
Applicable for school and college students to improve learning efficiency through personalized education.
